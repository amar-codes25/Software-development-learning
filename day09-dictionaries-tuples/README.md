# Day 09 — Dictionaries and Tuples

## What I'm covering today
- Key-value pairs and dictionary operations
- Tuples and immutability
- Choosing the right data structure for a problem

## Things to remember
- Use a dict when you need to look things up by a meaningful key
- Use a tuple when data shouldn't change (e.g. coordinates)
