# Day 10 — Errors, Exceptions, Debugging

## What I'm covering today
- Reading a traceback properly
- try/except and common exception types
- Using an IDE debugger / breakpoints

## Things to remember
- The traceback tells you exactly where and what — read it bottom to top
- Never randomly change code hoping it fixes itself — isolate the failure first
