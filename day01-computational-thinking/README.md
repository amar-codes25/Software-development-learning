# Day 01 — Computational Thinking

## What I'm covering today
- What computational thinking is: decomposition, pattern recognition, abstraction, algorithm design
- Breaking real-world problems into steps
- Flowcharts — reading and drawing them
- What makes a 'good' algorithm

## Things to remember
- Decomposition = breaking a big problem into smaller manageable chunks
- Abstraction = stripping away irrelevant detail, keeping only what matters
- These are general problem-solving skills, not just 'coding' skills
