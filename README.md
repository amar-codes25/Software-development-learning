# CMPC120 — Software Development (Roehampton, Level 4, 30 credits)

15-day intensive self-study repo. Goal: not just pass the module, but reach teaching-level mastery — able to answer any question on any topic covered here without hesitation.

**Assessment this is preparing for:**
- Computational problem-solving portfolio — 60%
- Live programming challenge with code review — 40%

## Structure

Each `dayXX-topic-name/` folder contains:
- `README.md` — what to learn that day, key things to remember, and what to practice
- `practice.md` — the practice tasks for that day with space for solutions
- `journal.md` — filled in *after* the day's work, written in your own words as if teaching someone else
- Any `.py` / `.c` / other code files written during that day's practice

## The rule

A day isn't done until the `journal.md` for that day can be read back and sound like a clear explanation to a complete beginner — no notes, no hedging. That's the actual bar, not "I followed along."
