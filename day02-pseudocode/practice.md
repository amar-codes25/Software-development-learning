# Day 02 Practice — Pseudocode

## Task 1
Write pseudocode for 3 everyday tasks (e.g. checking out at a shop, deciding what to wear based on weather)

* Tutoring session 
START

GET topic from schedule
MAKE notes on topic
CREATE teaching slides
PRACTICE topic
TEACH topic

WHILE student has doubts DO
    clear doubt
END WHILE

IF student needs help with school work THEN
    help student with school work
ELSE
    END tutoring session
END IF

END

* MMA training 
START

GET gear ready
TRAVEL to gym
WEAR gear
WARM UP

START training

WHILE training is not finished DO
    IF tired THEN
        REST for 30 seconds
    ELSE
        CONTINUE training
    END IF
END WHILE

IF have doubts THEN
    ASK coach
ELSE
    END training session
END IF

TRAVEL home

END

* Groceries picking 
START

LIST groceries needed
TRAVEL to supermarket

WHILE items remaining on list DO
    PICK next item from shelf
END WHILE

PAY at billing counter
TRAVEL home

END


## Task 2
Convert yesterday's tea-making steps into formal pseudocode

START

GET saucepan
ADD 1 cup of milk
ADD 1/4 cup of water
PLACE saucepan on stove
TURN stove on

WHILE not boiling DO
    wait
END WHILE

BOIL for 5 minutes
ADD 1 tbsp tea leaves
ADD 1/2 tbsp sugar
BOIL for 3 more minutes

IF tea is clear THEN
    pour into cup
ELSE
    strain then pour into cup
END IF

TURN stove off

END
